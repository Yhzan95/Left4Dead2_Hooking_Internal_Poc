# <p align="center">Left4Dead2 Internal Hooking Interface</p>
Left4Dead2 Hooking Internal is a C++ internal DLL project for the Old Source Engine. It demonstrates DLL injection, interface retrieval through CreateInterface, access to the engine’s ICvar interface, and colored logging directly in the in-game console.
This project is intended for educational and research purposes, such as learning about Source Engine interfaces, internal debugging, and basic DLL-based development.

## 🧑🏻‍💻 Usage
```js
1) Start Left4dead2
2) Inject the compiled DLL into the game process using Process Hacker or any DLL injector of your choice
3) Once the DLL has been injected successfully colored log messages will appear in the in-game developer console
```
